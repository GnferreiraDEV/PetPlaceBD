#PetPlace – Sistema de Gestão para Pet Shop
Trabalho Final – Disciplina de Banco de Dados
Universidade Católica de Brasília (UCB)

Este repositório apresenta o desenvolvimento do sistema PetPlace, projetado como Trabalho Final da disciplina de Banco de Dados da Universidade Católica de Brasília.
O objetivo principal foi aplicar conceitos de modelagem relacional, implementação de SGBD, integração com banco NoSQL e desenvolvimento de um sistema funcional para gestão de operações de um pet shop.
